# Neet-Mock-Test
A simple offline NEET mock test interface that allows students to practice without an internet connection and attempt any question of their choice, providing a flexible and convenient self-study experience.
# Neet Mock Test interface 
This project is a lightweight offline NEET mock test interface built using HTML and CSS. It allows students to practice mock tests without requiring an internet connection. Users can navigate freely and attempt any question of their choice, making the practice experience more flexible and convenient.
# Key features 
• Works completely offline
• No internet connection required
• Flexible question navigation
• Attempt questions in any order
• Simple and responsive interface for self-practice
